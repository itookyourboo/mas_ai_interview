# Мультиагентные системы. AI-собеседование

Мультиагентная система для проведения технических собеседований с использованием AI.

## Функции

- **Парсинг вакансий с hh.ru** — автоматическое извлечение информации о позиции, технологиях и требованиях
- **Генерация вопросов** — создание релевантных технических вопросов с помощью нескольких агентов
- **Оценка ответов** — многофакторная оценка с использованием 6 специализированных агентов
- **Веб-интерфейс** — удобный интерфейс для прохождения собеседования и просмотра результатов

## Архитектура агентов

### Агенты генерации вопросов
1. **Design Agent** — планирует структуру вопроса
2. **Generate Agent** — создаёт текст вопроса
3. **Validate Agent** — проверяет качество вопроса
4. **Format Agent** — форматирует вопрос в структурированный JSON

### Агенты оценки ответов
1. **Assessment Coordinator** — координирует оценку, запускает агентов, применяет веса
2. **Correctness Agent** — проверяет функциональную корректность и соответствие фактам
3. **Code Quality Agent** — анализирует стиль кода, читаемость, best practices
4. **Conceptual Understanding Agent** — оценивает глубину понимания концепций
5. **Relevance Agent** — проверяет релевантность ответа вопросу
6. **Level Alignment Agent** — оценивает соответствие уровню кандидата (Junior/Middle/Senior)
7. **Explanation Agent** — объединяет фидбек и формирует итоговую оценку

## Установка

### 1. Клонировать репозиторий

```shell
git clone <repo-url>
cd mas_ai_interview
```

### 2. Установить зависимости (с помощью uv)

```shell
uv sync
```

### 3. Настроить переменные окружения

Создайте файл `.env` на основе `.env.template`:

```shell
cp .env.template .env
```

Заполните переменные:

```env
MODEL_API_KEY=your_gigachat_api_key
MODEL_NAME=GigaChat-Max
MODEL_BASE_URL=  # опционально
```

## Запуск

### Веб-интерфейс (Streamlit)

```shell
uv run streamlit run src/app.py
```

Откройте в браузере: http://localhost:8501

### Генерация вопросов (CLI)

```shell
uv run src/main.py
```

Результат будет сохранён в `interview_questions.json`.

## Использование

### 1. Создание собеседования

1. Откройте веб-интерфейс
2. Выберите способ настройки:
   - **Ввод вручную** — укажите позицию, технологии, уровень, темы
   - **Загрузка с hh.ru** — вставьте ссылку на вакансию
3. Введите имя кандидата
4. Нажмите "Начать собеседование"

### 2. Прохождение собеседования

1. Отвечайте на вопросы в текстовом поле
2. Используйте кнопки навигации для перехода между вопросами
3. По завершении нажмите "Завершить"

### 3. Просмотр результатов

После оценки вы увидите:
- Общую оценку (0-10)
- Рекомендацию (сильный кандидат / требует развития / не соответствует)
- Детальную оценку по каждому вопросу
- Feedback от каждого агента
